Whisper 是一個 Android AI 角色扮演聊天 App。 支援自定义 API 接入，相容 OpenAI 格式，可以储存最多 10 套 API 设定随时切换，支援取得可用模型清单。 可以填寫自訂系統提示詞，建議搭配越獄提示詞使用效果更好。 內建記憶系統（RAG 語義檢索），支援 SillyTavern 角色卡匯入（V1/V2 格式），支援串流回應逐字輸出，聊天記錄儲存，深色模式，主動訊息推播。 病嬌模式下 AI 可以自主決定是否觸發手機控制功能，包括熄屏、清空相簿、查看前台應用程式等。 使用方法：下載專案編譯專案在手機上安裝編譯後的 APK，開啟設定頁，填寫 API Key 和 URL，填寫模型名稱或直接取得模型清單，可選填系統提示詞（建議填寫越獄提示詞以獲得更好的體驗），匯入角色卡後即可開始聊天。
